How it works

The pipeline, end to end

One command runs the whole chain locally on the developer's machine. Nothing is outsourced to third-party servers.

Pick a topic

A topic is selected from a curated, analytics-ranked pool.

Write the script

A short, hook-driven script is generated for the topic.

Render audio

TTS narration is rendered and mixed with licensed music.

Compose video

Visuals and word-synced captions are composited into a vertical video.

Publish

The finished video is uploaded to the developer's own TikTok account.

TikTok integration

How shorts-pipeline uses the TikTok API

shorts-pipeline integrates with Login Kit and the Content Posting API. It is a single-user tool: the only account it ever authenticates or posts to is the developer's own. It never accesses other users' data, never reads messages or followers, and never transmits TikTok data to any third-party server.

ScopeWhat it's used for
user.info.basic Reads the authenticated account's display name, avatar, and open ID — used only to confirm the pipeline is connected to the correct (developer-owned) account before posting.
video.upload Uploads the locally rendered video file to the developer's TikTok account as a draft via the Content Posting API.
video.publish Publishes the uploaded video directly to the developer's own TikTok profile with its generated title and hashtags.

OAuth tokens are stored only in a local environment file on the developer's machine and can be revoked at any time from TikTok's account settings. Full details are in the Privacy Policy.

FAQ

Common questions

Who can use shorts-pipeline?

Only the developer. shorts-pipeline is not a public service, has no sign-up, and posts exclusively to accounts owned by the developer. It exists to automate the developer's own content workflow.

What data does it collect from TikTok?

Only the authenticated developer account's basic profile info (display name, avatar, open ID) and the OAuth tokens needed to upload. No data about any other TikTok user is ever accessed, stored, or shared. See the Privacy Policy.

Is the content compliant with TikTok's guidelines?

Yes. All published content is reviewed against TikTok's Community Guidelines, uses only licensed music, and is generated by the developer's own pipeline — see the Terms of Service.

Is shorts-pipeline affiliated with TikTok?

No. shorts-pipeline is an independent tool and is not affiliated with, endorsed by, or sponsored by TikTok Ltd. or ByteDance Ltd.
